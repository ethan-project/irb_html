IACUC

※ 행정간사가 전문심사 건너뛰기를 실행하려는 경우, 화면 하단의 전문심사 건너뛰기를 실행하면 전문심사가 자동으로 마무리됩니다.
자동으로 작성된 심사 총평, 심사 결과(화면 하단에 위치)를 확인 또는 수정하고, 심사결과 제출동의를 체크하면 일반 심사가 바로 진행됩니다.
※ 전문심사 건너뛰기 기능은 행정간사일 경우에 활성화되므로, 행정간사가 실제 전문심사를 진행하는 중에 실행하지 않도록 주의하시기 바랍니다.
