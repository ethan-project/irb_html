사용자 등록

기관의 실험, 심사, 행정에 참여할 수 있는 사용자를 등록합니다.

사용자 신규 등록

  • 등록되는 사용자의 개인 정보 보호를 위해 최소한의 본인 확인이 가능한 정보로 임시 계정을 생성합니다.
  • 행정간사, 행정담당, 위원장, 심사위원, 연구원 등의 모든 권한을 등록할 수 있습니다.
  • 사용자의 Email과 휴대 전화 번호 등의 연락처를 모두 입력한 경우에만 등록이 가능합니다.
