등록 기관 정보

온라인 심사 시스템(IPSAP) 서비스에 등록된 기관 정보입니다.

서비스 이용 상태 :

※ 기관의 서비스 상태가 변경됩니다. 정확히 확인 후 변경해주세요.
이용 정지 :
기관의 모든 서비스가 중지되며, 기관의 사용자들도 본인의 사용자 정보만 조회가 가능합니다.
기관 계정 탈퇴 :
기관의 모든 서비스가 중지되며, 기관 자체의 계정과 기관의 모든 사용자 계정을 탈퇴 시킵니다.

연구기관 정보

* IACUC, IBC, IRB 등 위원회가 설치된 기관의 정보를 입력해 주세요.
* 기관의 영문명을 참조하여 대표 이니셜을 지정해 주십시오. 예) Hankuk University Medicine → “HUM” 입력
* 알파벳 대문자 + 숫자 조합으로 3~5자리로 지정 가능합니다.
* 중복확인 기능을 사용하여 동일한 코드가 있으면, 알파벳 대문자에 숫자를 부여하여 지정해 주시기 바랍니다. 예) HUM01, HUM02…
기본 주소
* 연구기관 로고는 .jpg, .png 등의 이미지 파일형식으로 첨부해 주십시오.
* 이미지 파일의 용량은 20MB, 사이즈는 219 pixel(가로) × 60 pixel(세로) 이하 이어야 합니다.

행정 업무 담당자 정보

* 동물실험계획의 심사절차와 서비스 이용료 정산 등의 행정업무를 수행하는 행정간사의 정보를 입력합니다.
* 입력한 개인정보는 심사와 관련된 행정절차 및 정보안내 외에는 사용하지 않습니다.
* 정보수신 동의를 해주셔야 심사절차와 관련된 안내가 가능합니다.

심사 유형

* 소속 연구기관에서 활용할 심사유형을 선택해 주세요.
* 소속 연구기관에 설치된 위원회를 모두 선택해 주세요.
점 이상
개 이상

결제 정보

* 소속 연구기관의 정산을 위한 기관정보를 입력해 주세요.
* 사업자등록증은 .pdf, .jpg, .png, .tiff 등의 이미지 파일형식으로 첨부해 주십시오.
* 파일의 용량은 20MB 이하 이어야 합니다.
* 해당되는 모든 결제방식을 선택해 주세요.
* 위의 "연간 계약" 결제 방식은 IPSAP와 연간 계약을 체결한 경우 선택해 주세요.
