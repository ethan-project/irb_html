Mail Template
저장된 Mail Template의 확인만 가능하며, 내용의 변경은 Template File을 직접 수정해야 됩니다.
    Template-01 [심사 진행 단계 안내]
    제목: [IPSAP] <IACUC ${신청서유형} - ${과제명}>의 ${진행단계} 안내입니다.
    Template-02 [심사 시작/미이행 안내]
    제목: [IPSAP] <IACUC ${신청서유형} - ${과제명}>의 심사 ${진행상태} 안내입니다.
    Template-03 [심사 종료 안내]
    제목: [IPSAP] <IACUC ${신청서유형} - ${과제명}>의 심사 ${진행상태} 안내입니다.
    Template-04 [심사 결과 안내]
    제목: [IPSAP] <IACUC ${신청서유형} - ${과제명}>의 심사 결과 안내입니다.
    Template-50 [IPSAP 기관 계정 등록 승인 안내]
    제목: [IPSAP] ${기관명}의 서비스 이용이 승인되었습니다.
    Template-51 [IPSAP 기관 계정 탈퇴 안내]
    제목: [IPSAP] ${기관명}의 IPSAP 기관 계정 탈퇴 안내입니다.
    Template-52 [IPSAP 기관 정보 변경 승인 안내]
    제목: [IPSAP] ${기관명}의 정보 변경이 승인되었습니다.
    Template-53 [IPSAP 사용자 계정 탈퇴 안내]
    제목: [IPSAP] IPSAP 사용자 계정 탈퇴 안내입니다.
    Template-60 [임시 비밀번호 발급 안내]
    제목: [IPSAP] ${대상자}님의 임시 비밀번호 발급 안내입니다.
    Template-61 [신규 계정, 사용자 등록 안내]
    제목: [IPSAP] IPSAP 계정 및 ${기관명}의 사용자 등록 안내입니다.
    Template-62 [기관 사용자 등록 안내]
    제목: [IPSAP] ${기관명}의 사용자 등록 안내입니다.
    Template-63 [기관 사용자 탈퇴 안내]
    제목: [IPSAP] ${기관명}의 사용자 탈퇴 안내입니다.
    Template-64 [기관 사용자 탈퇴 완료 안내]
    제목: [IPSAP] ${기관명}의 사용자 탈퇴 완료 안내입니다.
    Template-65 [기관 사용자 탈퇴 공지]
    제목: [IPSAP] ${기관명}의 사용자 탈퇴 공지입니다.
